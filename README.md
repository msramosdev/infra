# infra
This repository contains a Debian enviroment installer with the tools bellow:<br />
  * Google Chrome
  * Snap Store
  * DBeaver
  * Visual Studio Code
  * Docker
  * Docker Compose
  * Terminator
  * Postman
  * IntelliJ IDEA Community
  * GitKraken
  * etc...

<br />Check all packages at debian-environment.sh.<br /><br />
**Installing environment:**<br /><br />
$ sudo bash debian-environment-installer.sh
